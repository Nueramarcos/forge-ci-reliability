# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Forge CI Reliability, please report it privately by opening a security advisory at:

https://github.com/Nueramarcos/forge-ci-reliability/security/advisories/new

Please do **not** report security vulnerabilities through public GitHub issues.

## Supported Versions

Currently, the latest version of Forge CI Reliability is supported with security updates.
